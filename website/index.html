Installation

How to install

Manual installation takes about 60 seconds. You only need to do this once.

1

Download and unzip

Download the .zip above and unzip it to a folder you'll remember — like your Desktop.

2

Open extensions page

Type chrome://extensions into the address bar and press Enter.

3

Enable Developer Mode

Toggle on "Developer mode" in the top-right corner of the extensions page.

4

Load Unpacked

Click "Load unpacked" and select the extension/ folder. TrustAble appears in your toolbar.

FAQ

Common questions

Does it work on mobile?

TrustAble is a desktop browser extension. Mobile browser extensions aren't yet supported on most mobile browsers.

Does it scan pages automatically?

No — it only scans when you click the icon and choose to scan. It never runs in the background or monitors your browsing.

What AI does it use?

TrustAble uses Claude by Anthropic to analyze content. Results are streamed back in real time.

Is it free?

Yes — free to download and use. No hidden fees, subscriptions, or premium tiers.
